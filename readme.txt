Steps to use application-(Note-This steps to be followed when the website url is not working)

1)Download and unzip the folder 
2)After that open the folder is Vscode or any developing software as pycharm.
3)Go to main.py
4)Open terminal and type "python main.py"and press enter
5)After that server will start running, then press ctrl and point the pointer on localhost address.
6)Then the application will be ready to use.
